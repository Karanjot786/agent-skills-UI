### `context`

Generate system prompt context for AI agents in multiple formats.

```bash
skills context [options]
```

| Option | Description |
|--------|-------------|
| `-f, --format <format>` | Output format: `xml`, `json`, `markdown` (default: `xml`) |
| `-s, --skills <skills...>` | Specific skills to include (default: all) |
| `-o, --output <file>` | Write to file instead of stdout |

**Examples:**
```bash
skills context                           # XML to stdout
skills context -f json                   # JSON format
skills context -f markdown -o context.md # Save markdown to file
skills context -s xlsx,pdf               # Specific skills only
```

### `preview`

Open skill detail page in browser or print URL.

```bash
skills preview <skill-name> [options]
```

| Option | Description |
|--------|-------------|
| `--url-only` | Just print the URL without opening |

**Example:**
```bash
skills preview xlsx            # Opens in browser
skills preview xlsx --url-only # Prints: https://skills.karanjot.dev/marketplace/xlsx
```

### `completion`

Generate shell completion script.

```bash
skills completion <shell>
```

**Supported shells:** `bash`, `zsh`, `fish`

**Example:**
```bash
# Add to ~/.bashrc
eval "$(skills completion bash)"

# Add to ~/.zshrc
eval "$(skills completion zsh)"

# Save for fish
skills completion fish > ~/.config/fish/completions/skills.fish
```
